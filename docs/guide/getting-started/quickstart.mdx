# Quick Start

## Requirements

<Tabs>
  <Tab label="Windows">
    Windows 10 or newer is required. Only Intel and AMD CPUs are supported (no ARM CPUs).

    **Supported GPUs:**

    - NVIDIA - https://developer.nvidia.com/cuda-gpus
    - AMD (via ZLUDA) - https://rocm.docs.amd.com/projects/install-on-windows/en/latest/reference/system-requirements.html
    - AMD/Intel Arc (via DirectML) - any GPU supporting DirectX 12 or newer

    To use ZLUDA, please follow the instructions here: [AMD GPUs on Windows via ZLUDA](/guide/getting-started/platform-specific-instructions.html#amd-gpus-on-windows-via-zluda)

  </Tab>
  <Tab label="macOS">
    macOS 10.9 or newer is required. Both Intel and Apple Silicon are supported.
  </Tab>
  <Tab label="Linux">
    glibc is required.

    **Supported GPUs:**

    - NVIDIA (via CUDA) - https://developer.nvidia.com/cuda-gpus
    - AMD (via ROCm) - https://rocm.docs.amd.com/projects/install-on-linux/en/latest/reference/system-requirements.html

  </Tab>
</Tabs>

## Installation

<Steps>
### Download Metastable

Download the installer from: https://metastable.studio/download

### Install dependencies

After installing Metastable and confirming your device meets the minimum requirements, you will be prompted to download the dependencies. This step may download up to 5GB.

Please be patient, this might take a while.

### Done

You're ready to generate images!

</Steps>
