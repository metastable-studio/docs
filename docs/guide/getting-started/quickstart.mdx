# Quick Start

## Requirements

<Tabs>
  <Tab label="Windows">
    Windows 10 or newer is required. Only Intel and AMD CPUs are supported (no ARM CPUs).

    **Supported GPUs:**

    - **NVIDIA**
        - List of supported GPUs: https://developer.nvidia.com/cuda-gpus
        - NVIDIA drivers must be installed: https://www.nvidia.com/en-us/drivers/
    - **AMD** (via ZLUDA)
        - List of supported GPUs: https://rocm.docs.amd.com/projects/install-on-windows/en/latest/reference/system-requirements.html
        - AMD GPU drivers and HIP SDK 5.7 or 6.1 must be installed.
        - Detailed installation instructions: [AMD GPUs on Windows via ZLUDA](./platform-specific-instructions.html#amd-gpus-on-windows-via-zluda)
    - **AMD/Intel Arc** (via DirectML)
        - Any GPU supporting DirectX 12 or newer is supported.
        - DirectML will usually have lower performance than the other methods. It's not recommended to use the DirectML mode.

    In case of no compatible GPUs being present, Metastable will run in a CPU-only mode.

  </Tab>
  <Tab label="macOS">
    macOS 10.9 or newer is required. Both Intel and Apple Silicon are supported.

    External GPU support is not available.

  </Tab>
  <Tab label="Linux">
    glibc is required.

    **Supported GPUs:**

    - **NVIDIA** (via CUDA)
        - List of supported GPUs: https://developer.nvidia.com/cuda-gpus
        - Proprietary NVIDIA drivers must be installed. The `nvidia-smi` command must be present in your system and within PATH.
    - **AMD** (via ROCm)
        - List of supported GPUs: https://rocm.docs.amd.com/projects/install-on-linux/en/latest/reference/system-requirements.html
        - AMD GPU drivers and ROCm HIP SDK must eb installed. The `rocm-smi` command must be present in your system and within PATH.

    In case of no compatible GPUs being present, Metastable will run in a CPU-only mode.

  </Tab>
</Tabs>

## Installation

<Steps>
### Download Metastable

Download the installer from: https://metastable.studio/download

### Install dependencies

After installing Metastable and confirming your device meets the minimum requirements, you will be prompted to download the dependencies. This step may download up to 5GB.

Please be patient, this might take a while.

### Done

You're ready to generate images!

</Steps>
